This folder contains video avatars for the game hostesses.

Current videos are placeholders downloaded from public sample repositories (Intel IoT DevKit).

To upgrade to "Real Human Avatars":
1. Use AI Video Generation tools like:
   - HeyGen (https://www.heygen.com/)
   - Synthesia (https://www.synthesia.io/)
   - D-ID (https://www.d-id.com/)
   
2. Generate a "Portrait" mode video (9:16 or 1:1 aspect ratio is best).
3. Ensure the video is a smooth loop (if possible) or a talking sequence.
4. Upload the video via the Developer Panel -> Edit Hostess page.
   OR
   Manually replace the files in this folder:
   - jasmin_avatar.mp4
   - layla_avatar.mp4
   - ruby_avatar.mp4
   - sarah_avatar.mp4

Note: The system supports MP4 format.
